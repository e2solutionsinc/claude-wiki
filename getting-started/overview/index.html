    Overview — Which Tool to Use

    All three Claude tools use the same account and subscription. They're not competing options most developers end up using all three depending on the task.

    Quick decision guide

    Use Claude.ai Web when:

    • You want a quick answer or a code snippet without opening your project
    • You're exploring an approach before writing any code
    • You need to share a conversation with a teammate
    • You want to upload a document or image for Claude to analyze

    Use Claude Code (CLI) when:

    • You want Claude to work directly inside your codebase
    • You're debugging, refactoring, writing tests, or reviewing changes
    • You want Claude to read files, make edits, and run shell commands for you
    • You're doing anything that involves your actual project files

    Use Claude Cowork when:

    • The task has multiple steps and you don't want to manage it step by step
    • You need Claude to produce output files (spreadsheets, docs, reports)
    • You want to kick something off and come back to finished work
    • You want to schedule a task to run on a recurring basis

    Account requirements

    All three tools require a paid Claude subscription. Check with your manager if you're unsure which plan you're on. The free Claude.ai plan does not include Claude Code or Cowork access.

    Plan Web Claude Code Cowork
    Free Limited No No
    Pro Yes Yes Yes
    Max Yes Yes Yes
    Team / Enterprise Yes Yes Yes
